Sound Files Directory

This directory contains sound files for the PennyWise app.

Required Files:
- kaching.mp3: Sound played when a new expense is added

To add the sound file:
1. Download or create a "kaching" sound effect
2. Name it "kaching.mp3" (or .ogg, .wav)
3. Place it in this directory
4. Delete this readme file

Sound requirements:
- Duration: 1-2 seconds
- File size: Under 100KB
- Format: MP3, OGG, or WAV
- Content: Pleasant "kaching" or cash register sound
